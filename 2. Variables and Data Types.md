Go is a **statically typed** language, meaning variable types are checked at compile time, not runtime. While this sounds strict, Go makes declaring variables incredibly simple and flexible.

### 1. Basic Declaration (`var`)

The traditional way to declare a variable. Note that the type comes **after** the variable name.

Go

```
package main

import "fmt"

func main() {
    // Declaring a variable with an explicit type (string)
    var language string = "Go" 
    
    // Declaring a variable, letting Go infer the type (int)
    var version = 1.22 
    
    // Declaring a zero-value variable (string defaults to "")
    var name string 
    
    name = "Gopher"
    
    fmt.Println(language, version, name) // Output: Go 1.22 Gopher
}
```

### 2. The Short Declaration Operator (`:=`)

This is the most common way to declare and initialize a variable inside a function. The `:=` operator is a **shorthand** for `var name = value` and the type is always inferred by Go.

**ðŸ”¥ Important:** You can only use `:=` for **new** variables; you cannot use it to re-assign an existing variable.

Go

```
package main

import "fmt"

func main() {
    // Short declaration for a new variable
    message := "Hello from := " 
    
    // This is valid:
    message = message + "!" 
    
    // This would cause a compile error:
    // message := "Error!" 
    
    // Declaring multiple variables at once
    a, b := 10, true 
    
    fmt.Println(message) // Output: Hello from := !
    fmt.Println(a, b)    // Output: 10 true
}
```

### 3. Core Data Types

Go's basic types are clean and predictable:

|**Type Category**|**Specific Types**|**Description**|**Example**|
|---|---|---|---|
|**Numeric**|`int`, `int32`, `int64`, `float32`, `float64`|Integers and floating-point numbers. `int` usually matches the architecture (32 or 64 bit).|`var i int = 42`|
|**Boolean**|`bool`|`true` or `false`.|`var isTrue bool = true`|
|**String**|`string`|UTF-8 encoded sequence of characters.|`var s string = "Golang"`|

#### Example: Type Inference

Go is excellent at figuring out the type for you:

Go

```
package main

import "fmt"

func main() {
    x := 10        // Type inferred: int
    y := 3.14      // Type inferred: float64
    z := "example" // Type inferred: string
    
    // Use fmt.Printf and %T to see the type
    fmt.Printf("x is of type %T\n", x) 
    fmt.Printf("y is of type %T\n", y) 
    fmt.Printf("z is of type %T\n", z) 
}
/*
Output:
x is of type int
y is of type float64
z is of type string
*/
```
