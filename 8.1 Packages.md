A **package** is the fundamental unit of code organization and reuse in Go. It's essentially a directory containing one or more `.go` source files. Every Go file must belong to a package.

### Package Declaration and Types

The package name is declared at the top of every file. It defines the namespace for the code within that directory.

- **`package main`**: This is a special package name used exclusively for the entry point of an **executable program**. It must contain the `func main()`.
    
- **Library Packages**: Any other name (e.g., `net/http` is the `http` package, `fmt` is the `fmt` package) defines a reusable library. These packages do not have a `main` function.
    

### Visibility (Exporting)

Go has a simple, rigid rule for visibility that determines if an identifier (function, variable, type, or constant) is accessible from _outside_ its own package:

|**Rule**|**Example**|**Visibility**|**Rationale**|
|---|---|---|---|
|**Exported**|`Add`, `UserType`|**Public** (Accessible from _other_ packages).|The identifier starts with a **Capital Letter**.|
|**Unexported**|`calculateValue`, `tempVar`|**Private** (Only accessible within its _own_ package).|The identifier starts with a **lowercase letter**.|

This simplicity removes the need for `public`, `private`, or `protected` keywords.

#### Example: Importing and Accessing

When importing a package, you access its **exported** identifiers using dot notation (`package.Identifier`).

Go

```
package main 

import (
	"fmt"
	"math" // Importing the math library package
)

func main() {
	// math.Abs is EXPORTED (starts with 'A')
	result := math.Abs(-25.0) 
	fmt.Println("Absolute value:", result) 
	
	// math.pi is UNEXPORTED (starts with 'p')
	// fmt.Println(math.pi) // Compile ERROR: Cannot access unexported identifier
}
```

### Custom Imports and Aliasing

You can import packages and give them a local alias, or use a blank identifier (`_`) for side effects.

|**Import Format**|**Example**|**Purpose**|
|---|---|---|
|**Alias**|`import m "math"`|Access the package via `m.Abs()` instead of `math.Abs()`.|
|**Dot**|`import . "fmt"`|Makes all exported identifiers directly accessible without the package name (`Println("Hi")`). **Discouraged** in production code as it reduces clarity.|
|**Blank**|`import _ "github.com/lib/pq"`|Imports a package solely for its side effects (like database driver registration). The package identifiers are **not** accessible.|
